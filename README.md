# 버전 관리 시스템(VCS:Version Control System)

### 깃(git) 
 * 컴퓨터 파일의 변경사항을 추적하여 여러 명의 사용자들 간에 해당 파일들의 작업을 조율하기 위한 분산 버전 관리 시스템 


### 소스 관리는 어떻게?
 * 개발할 때 소스 관리는?
   * 각 폴더는 무슨 버전인가?
   * 왜, 언제 수정했지?
   * 1년 후에 보면 무슨 소스인지 알까?
   * 어제 개발한 거 다시 가져오려면...
 
 * 협업할 때 소스 관리는?
   * 압축해서 Gmail로?
   * Drive를 이용해서 share?
   * 서버에 올려놓고 같이 접속해서 고치면?
   * 최종본은 누구?
