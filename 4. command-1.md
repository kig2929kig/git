### git 상태 확인
   ```linux
   # git status
   ```

### staging area 저장
   ```linux
   # git add 파일명
   ```
### local Repo 저장
   ```linux
   # git commit -m "주석"
   ```

### log 확인
   ```linux
   # git log
   # git log -3
   # git log -p -1 //이전 버전과 차이를 함께 보여줌
   # git log -2 --oneline 
   # git log --author=kig2929kig 
   # git log --pretty=oneline -graph
   ```

   ```linux
   # git rm --cached 파일명
   ```
